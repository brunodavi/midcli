# midcli

Commandline to image generate from Midjorney on Hugface

## Usage

```
NAME
    main.py

SYNOPSIS
    main.py PROMPT <flags>

POSITIONAL ARGUMENTS
    PROMPT
        Type: str

FLAGS
    -s, --style=STYLE
        Default: 'Anime'
    -f, --filename=FILENAME
        Default: 'image'
    -i, --image_dir=IMAGE_DIR
        Default: '.'

NOTES
    You can also use flags syntax for POSITIONAL ARGUMENTS
```
